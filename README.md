# <p align="center">🌸 Horoscope Fortune Teller 🌸</p>

## Practice: Bringing together everything we've learned so far

### Understanding JS -> Combining variables, conditionals, and random numbers to create something magical

<hr>

## <p align="center">💫 Overview</p>
-   [ ] This is where everything starts to come together. Variables taught me how to store information. Conditionals taught me how to make decisions. Math.random() taught me how to add a little unpredictability. This lead to combining everything into an actual program, a fortune teller that knows your zodiac sign and shares a little wisdom based on the roll of a digital die. It is simple, but it has a little personality.

<hr>

## <p align="center">🎨 Key Design Features</p>
-   [ ] <b>Variables</b>
    -   [ ] Storing the horoscope month, zodiac sign, random number, and fortune message

-   [ ] <b>Control Flow (if / else if / else)</b>
    -   [ ] Determing which zodiac sign matches the given month
    -   [ ] Selecting a random fortune based on a generated number

-   [ ] <b>Math.random()</b>
    -   [ ] Adding unpredictability so each run feels unique

-   [ ] <b>String Concatenation</b>
    -   [ ] Comnbining text, variables, and line breaks into a readable fortune

-   [ ] <b>Console Output</b>
    -   [ ] Displaying the final fortune to the user

<hr>

-   [ ] Simple practice to understand that programming is not just about learning isolated concepts. It is about weaving them togehter into something meaningful
    -   [ ] Variables hold the data that matters
    -   [ ] Conditionals make choices about what to do with that data
    -   [ ] Random numbers add surprise and delight
    -   [ ] console.log() shares the result with the world
